game.py 112-113行目にて、カメラの番号を指定する必要があります。
カメラを二つしか接続していない場合Windowsなら0,1ですがMacだと1,2だと思われます。
例
'source': 0,
 'sub_source': 1,

color_game.py 51行目においても同様です。

起動した際、カメラの起動が環境によって時間がかかる場合があります。
また、シューティングゲームが終わった直後、カメラのウィンドウを終了せずに進めると、Windowsの場合ゲームがクラッシュします。
必ず進める前にカメラのウィンドウにqキーを入力して画面を終了させてください。長押ししないと終了しない場合があります。
